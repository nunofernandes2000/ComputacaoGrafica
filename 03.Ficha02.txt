Considering the implementation made in Ficha01, where several random graphical forms are drawn, implement the following functionalities:
1. Change the program to allow the world coordinates to be the same as the screen coordinates.
2. Each form (Bounding Boxes/rectangles) has a fill and limit color, which is randomly generated when the form is created and does not change.
3. Each form has a type that defines its shape. Implement at least the following types, generating random forms of these types:
  . Rectangle;
  . Square;
  . Isosceles triangle;
  . Line.
4. Implement a functionality that whenever the counter (from the display callback) is even, one of the forms is deleted from memory (randomly); When there are no more forms create random forms again.
5. Also implement the following types of forms:
  . Equilateral triangle;
  . Hexagon;
  . Circumference.
6. Investigate a way to use the mouse to create a random form; when the mouse click is detected it creates a random form.
